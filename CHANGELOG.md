# 更新日志

## [2.0.0] - 2025-12-05

### 🎉 重大更新：摄影相册系统

#### 新增功能
- **摄影作品相册系统**
  - 一个作品可包含多张照片（普通用户10张，管理员50张）
  - 每张照片支持独立的EXIF参数（相机、镜头、焦段、光圈、快门、ISO）
  - 相册级别元数据（拍摄地点、拍摄日期）
  - 每日发布配额限制（3个相册/天）

- **瀑布流布局**
  - 作品列表采用 Pixabay 风格的瀑布流展示
  - 响应式网格布局
  - 类型筛选（全部/项目/摄影）
  - 图片懒加载

- **图片轮播详情页**
  - 大图轮播展示
  - 缩略图导航
  - 照片切换时参数同步更新
  - 左右分栏布局（照片区70% + 信息栏30%）

- **用户作品管理**
  - 我的作品列表（`/dashboard/works`）
  - 创建/编辑作品
  - 多图上传和参数编辑
  - 实时配额显示

#### 后端改进
- 新增 `PhotoItem` 和 `PhotoItemMetadata` 数据模型
- 照片数量限制验证（基于用户角色）
- 每日配额检查逻辑
- 作品类型筛选API
- 权限控制（作者/管理员）

#### 前端改进
- 完整的摄影作品编辑界面
- 折叠面板展示照片列表
- 每张照片独立参数输入
- 瀑布流布局组件
- 图片轮播组件
- 响应式设计（移动端适配）

#### API 变更
- `POST /api/works` - 支持 PhotoItem 数组
- `PUT /api/works/:id` - 支持 PhotoItem 数组
- `GET /api/works` - 新增 `type` 参数筛选
- `GET /api/works/my` - 获取用户作品列表
- `GET /api/works/quota` - 查询配额使用情况

#### 数据结构变更
- `Work.images` 字段：从 `[]string` 改为 `[]PhotoItem`
- `Work.metadata` 字段：支持相册级别元数据
