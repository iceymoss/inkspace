# 风格 C · 温暖手作感「小屿的角落」设计规范

> 原型：`style-c-cozy.html` · 默认浅色模式
> 关键词：暖纸色、拍立得、胶带、手绘线、圆角

## 1. 设计原则

这一版把网站做成一间去朋友家做客的小屋：文案用口语（「家」「进来坐坐」「亲手做的东西」），照片是贴在墙上的拍立得，栏目像三本摊开的笔记本。物理世界的质感是核心隐喻——胶带、邮票、图钉、纸张的旋转角度——但全部用 CSS 实现，不用位图贴图。圆角刻意不规则（四角半径各不相同），旋转角度控制在 ±7° 以内：歪一点是手作，歪太多是混乱。配色避开常见的奶油+赤陶组合，用苔绿 + 芥黄 + 灰蓝 + 藕粉的组合建立自己的暖色系。

## 2. 色彩令牌

| Token | 浅色模式 | 深色模式 | 用途 |
|---|---|---|---|
| `--bg` | `#FBF6ED` | `#26201A` | 页面背景（暖纸/暖夜） |
| `--card` | `#FFFDF8` | `#2E2721` | 卡片底 |
| `--ink` | `#43362A` | `#EFE6D9` | 正文（暖褐墨） |
| `--sub` | `#93826F` | `#A79681` | 次要文字 |
| `--line` | `#EADFCD` | `#3D342B` | 描边（2px 为主） |
| `--moss` | `#75845C` | `#A3B385` | 主强调（苔绿）：激活、悬停、品牌 |
| `--moss-soft` | `#EDEFE2` | `#333A29` | 苔绿软底（胶囊、标签） |
| `--mustard` | `#D9A441` | `#E0B25C` | 次强调（芥黄）：主按钮、圆点、胶带 |
| `--sky` | `#8FAABF` | `#93B0C6` | 辅助（灰蓝）：分类色 |
| `--blush` | `#E5B8A5` | `#D9A891` | 辅助（藕粉）：分类色 |
| `--shadow` | `0 6px 24px rgba(122,98,70,.10)` | `0 6px 24px rgba(0,0,0,.35)` | 软暖阴影 |

分类三色系统：设计=苔绿、代码=灰蓝、摄影=藕粉，贯穿标签、图标底、笔记本书脊。

## 3. 字体系统

| 角色 | 字体栈 | 使用位置 |
|---|---|---|
| 正文/标题 | `'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif` | 全站 |
| 圆体 `.round`（可选增强） | `'Yuanti SC', 'PingFang SC', …` | 品牌、大标题（有则更软） |

字号阶梯：

| 层级 | 字号 | 行高 | 字重 |
|---|---|---|---|
| H1 Hero | `clamp(32px, 5vw, 48px)` | 1.4 | 800 |
| H2 栏目 | 24px | — | 800 |
| H3 卡片 | 17–17.5px | 1.55 | 700 |
| 正文 | 15.5px | 1.85 | 400 |
| 辅助 | 12.5–13.5px | — | 400 |

本风格标题字重最重（700–800），靠重而圆的字形传递憨厚感；行高整体比其他风格更松。

## 4. 间距、圆角与形状语言

| Token | 值 |
|---|---|
| 内容最大宽度 | 1020px，两侧 28px |
| 区块垂直 padding | 56px（Hero 72px/64px） |
| 卡片圆角 | 20px；笔记本 `8px 20px 20px 8px`（书脊侧小圆角） |
| 不规则圆角（logo/图标底） | 如 `12px 14px 13px 15px`、`16px 18px 15px 19px` |
| 有机圆点 | `border-radius: 40% 60% 55% 45%` |
| 描边 | 卡片 2px 实线；徽章 1.5px 虚线；邮票 2px dashed |
| 旋转角度 | 徽章/logo ±1.5–4°；拍立得 ±2–7°；邮票 -8° |
| 网格 gap | 18–20px |

## 5. 组件规范

**导航**：sticky，高 68px，90% 透明 blur。品牌 = 不规则圆角方块 logo（苔绿底白字，旋转 -4°）+ 站名。链接为胶囊：悬停 `--moss-soft` 底，激活苔绿底白字。主题按钮圆形 2px 描边，悬停旋转 20° 且描边变芥黄，图标随模式切换 ☾/☀。

**Hero（左文右拍立得）**：问候胶囊（`--moss-soft` 底，旋转 -1.5°）→ H1（一个词用手绘波浪线 SVG 下划线，一个词用芥黄→藕粉渐变文字）→ 简介 → 双 CTA。主按钮「果酱按钮」：芥黄底胶囊 + `box-shadow: 0 4px 0 <深芥黄>` 实体底边，按下时下移 2px、底边变 2px（物理按压感）。次按钮为点状虚线下划线文字。

**手绘波浪分隔线**：SVG `<path d="M0 13 Q 30 0 60 13 T 120 13 …">` 平铺，`stroke: currentColor`（取 `--line` 色），高 26px，放在 Hero 与正文区之间。

**随笔卡片（博客）**：3 列，`--card` 底 + 2px 描边 + 20px 圆角。结构：分类胶囊（三色系统）→ 标题（flex:1 撑开）→ 摘要 → 日期+阅读时长。悬停：上移 5px + 旋转 -0.5° + 描边变苔绿 + 软阴影。

**手作架（项目）**：3 列卡片。emoji 图标放在不规则圆角色块中（旋转 -3°，底色按卡片序号轮换三色）。角色行用芥黄小字。悬停描边变芥黄。

**照片墙**：4 列拍立得：白底（深色模式 `#3A332B`）、`padding: 10px 10px 34px`（下宽边）、图注居中手写感。奇数张旋转 -2°、偶数张 +1.6°。悬停回正并 `scale(1.05)`、`z-index` 提升。过渡曲线 `cubic-bezier(.3,1.4,.5,1)`（轻微过冲，果冻感）。

**笔记本（知识库）**：3 列，左侧 12px 实色「书脊」（三色轮换），圆角书脊侧 8px、开口侧 20px。徽章为 1.5px 虚线胶囊。悬停上移 + 轻旋转。

**页脚**：居中布局：圆形虚线邮票（芥黄描边，「屿 · 制」，旋转 -8°）→ 链接行 → 一句话（「慢慢做，比较快」）。

## 6. 签名元素：胶带拍立得堆

Hero 右侧 380px 高的相对定位容器，三张拍立得绝对定位叠放（旋转 -6°/3°/-2°，z-index 1/2/3）。每张：白底 `padding: 12px 12px 40px`，顶部胶带条（74×26px，半透明芥黄纸色，旋转 -3°，`top: -12px` 骑缝）。悬停：回正 + `scale(1.06)` + z-index 9。照片位为 200×200 渐变占位，落地换实图。

## 7. 动效

| 场景 | 参数 |
|---|---|
| 主题切换 | `.35s` |
| 卡片悬停 | `.25s`，位移+旋转+描边 |
| 拍立得回正 | `.3–.35s cubic-bezier(.3,1.4,.5,1)`（过冲） |
| 果酱按钮按压 | `.15s`，translateY + shadow 高度互换 |
| 主题按钮 | 旋转 20° |

动效性格：Q 弹、有物理感，但频次低——只在悬停和按压时发生。

## 8. 响应式

| 断点 | 变化 |
|---|---|
| ≤900px | Hero 单列，拍立得堆居中（max-width 420px）；卡片类 2 列；照片墙 2 列；导航收起 |
| ≤560px | 卡片类 1 列 |

## 9. 无障碍

焦点样式用 2px 虚线苔绿描边（与手作语言一致）。浅色模式 `--ink #43362A` 对比度 ≈ 9:1；`--sub #93826F` ≈ 3.2:1，只用于辅助信息不用于正文。芥黄按钮文字用深褐 `#4A3814`（≈ 7:1）而非白色。装饰旋转元素不影响阅读顺序；reduced-motion 时取消回正/过冲动画。

## 10. Vue 落地提示

`PolaroidStack.vue`（props: photos[3]，slot 可扩展）、`PolaroidItem.vue`（props: rotate, caption）、`JamButton.vue`（按压态封装）、`WavyDivider.vue`（SVG 平铺，props: color）、`NotebookCard.vue`（props: spineColor）。三色分类系统集中为 `categoryColors = { design: 'moss', code: 'sky', photo: 'blush' }` 映射。旋转角度用 props 传入而非写死，保证照片墙可从 CMS 数据渲染。
