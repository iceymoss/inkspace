# 风格 A · 极简杂志风「屿刊」设计规范

> 原型：`style-a-magazine.html` · 默认浅色模式
> 关键词：纸感、留白、衬线、发丝线、竖排

## 1. 设计原则

这一版把网站当作一本正在连载的个人杂志：内容是主角，界面退到纸的角色。装饰全部来自排版本身——衬线字的笔画、发丝分隔线、大面积留白与一处竖排文字。任何新增元素前先问：它是在帮助阅读，还是在打扰阅读？没有卡片阴影，没有大色块，层级全靠字号、字重与间距表达。强调色黛绿只出现在少数关键位置（品牌字、悬停态、期刊号），出现得越少越贵重。

## 2. 色彩令牌

| Token | 浅色模式 | 深色模式 | 用途 |
|---|---|---|---|
| `--bg` | `#FAF9F6` | `#171614` | 页面背景（纸色） |
| `--bg-soft` | `#F2F0EA` | `#201F1C` | 次级背景（备用） |
| `--ink` | `#1B1A18` | `#EAE7E0` | 正文/标题墨色 |
| `--sub` | `#6E6A62` | `#98938A` | 次要文字、日期、注释 |
| `--hairline` | `#E2DFD6` | `#2D2B27` | 发丝分隔线、描边 |
| `--accent` | `#33544A` | `#8FB3A5` | 强调色（黛绿），悬停/期刊号/品牌 |
| `--accent-soft` | `#E4EBE7` | `#24312C` | 强调色底（备用） |

使用规则：主按钮为墨色底 `--ink` + 纸色字，悬停变 `--accent`；正文链接默认不带下划线，靠悬停变色提示；`--accent` 在整页出现不超过 5 处。

## 3. 字体系统

| 角色 | 字体栈 | 使用位置 |
|---|---|---|
| 展示衬线 `.serif` | `Georgia, 'Songti SC', 'Noto Serif SC', 'SimSun', serif` | H1/H2、品牌、序号、竖排 |
| 正文无衬线 | `'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif` | 正文、导航、按钮 |
| 小标签 `.label` | 同正文 | 眉题、栏目英文名 |

字号阶梯（桌面）：

| 层级 | 字号 | 行高 | 字重 | 其他 |
|---|---|---|---|---|
| H1 Hero | `clamp(40px, 6.5vw, 72px)` | 1.25 | 500 | `letter-spacing: .02em` |
| H2 栏目 | 26px | — | 500 | `.06em`，后缀英文 12px `.2em` |
| H3 条目 | 19px | 1.5 | 500 | — |
| 正文 | 16px | 1.8 | 400 | — |
| 辅助 | 13–14px | — | 400 | 日期 `.05em` |
| 标签/眉题 | 11–12px | — | 400 | 大写，`.2em–.3em` |

要点：衬线字重最多用到 500/600，不用 700+（保持杂志的轻）；中文标题字距整体 +.02em~.06em 更透气。

## 4. 间距与栅格

| Token | 值 |
|---|---|
| 内容最大宽度 | 1060px，两侧 padding 32px |
| 区块垂直 padding | 76px（Hero 96px 上 / 88px 下） |
| 栏目标题下间距 | 44px |
| 列表条目 padding | 26–30px 上下 |
| 网格 gap | 20px（照片、知识库卡片） |

分区之间用 1px `--hairline` 顶边线分隔，这是页面最主要的结构语言。

## 5. 组件规范

**导航**：sticky 顶栏，高 64px，背景为 88% 透明度纸色 + `backdrop-filter: blur(10px)`，底部 1px 发丝线。链接 14px 灰色，悬停/激活变墨色并出现 1px 黛绿下边线。右侧圆形主题切换按钮 34px，1px 发丝描边，悬停旋转 15° 并变绿描边。

**Hero**：期刊号行（`VOL.24` 黛绿衬线 + 64px 短线 + 大写眉题）→ 大标题（关键词用 `--accent` 着色）→ 简介（最长 34em，灰色）→ 双 CTA（实心墨色按钮 + 下划线幽灵链接）。右侧为签名竖排。

**文章列表**：三栏 grid `120px 1fr auto`（日期 / 标题+摘要 / 标签胶囊），条目间 1px 发丝线。悬停：整行左移 10px（`padding-left` 过渡），标题变绿。标签为 1px 描边胶囊，`border-radius: 99px`。

**项目列表**：三栏 grid `56px 1fr auto`（汉字序号 一/二/三 / 名称+简介 / 箭头）。序号默认发丝色，悬停变绿；箭头悬停右移 6px。

**摄影网格**：3 列，`aspect-ratio: 4/5` 基准，第 2 张拉高（4/6.2）、第 5 张上移 -38px 制造错落。悬停图片 `scale(1.045)`（600ms），底部渐变遮罩浮出图注（12px，字距 .18em）。图片上叠加 5px 点阵颗粒层（`opacity .35, mix-blend-mode: overlay`）模拟胶片感。

**知识库卡片**：3 列，1px 发丝描边、无阴影、无圆角，padding 34px 30px。结构：大写小标签（黛绿）→ 标题 → 描述 → 计数行。悬停：描边变绿 + 上移 4px。

**页脚**：顶边发丝线，左侧大号衬线站名 + 一句话，右侧 13px 链接组。

## 6. 签名元素：竖排文字 + 期刊号

竖排使用 `writing-mode: vertical-rl`，15px，字距 .55em，左侧 1px 发丝线 + 22px padding，高 300px，内容为站点态度语（如「记录 · 观察 · 创造 · 分享」），其中一个词用 `--accent` 加重。≤860px 隐藏。期刊号体系可延伸至文章详情页（每篇文章 = 一期）。

## 7. 动效

| 场景 | 参数 |
|---|---|
| 颜色/主题切换 | `all .35s ease`（body 背景与文字色） |
| 悬停位移（行、卡片） | `.25s` 默认缓动 |
| 图片缩放 | `.6s cubic-bezier(.2,.6,.2,1)` |
| 图注浮现 | `.35s`，`translateY(6px) → 0` + 透明度 |

整版无入场动画、无滚动触发动画——克制即风格。`prefers-reduced-motion` 时全部关闭。

## 8. 响应式

| 断点 | 变化 |
|---|---|
| ≤860px | 竖排隐藏；导航链接收起（落地换汉堡）；照片/知识库 2 列；文章列表单列堆叠；错落上移取消 |
| ≤560px | 照片/知识库 1 列 |

## 9. 无障碍

焦点样式：`outline: 2px solid var(--accent); outline-offset: 3px`。黛绿 `#33544A` 在纸色上对比度 ≈ 7.5:1，可安全用于小字；深色模式的 `#8FB3A5` 仅用于大字与非文本元素，小字仍用 `--ink`。

## 10. Vue 落地提示

主题作为全局 `data-theme`，token 写入 `:root` 层。组件对应：`PostRow`（三栏 grid 属性化）、`PhotoFigure`（props: ratio, offset）、`WikiCard`。颗粒层与渐变遮罩封装为 `PhotoFigure` 内部元素。衬线标题封装 `<TitleSerif>` 或全局 `.serif` 原子类。
